#  Event - Платформа мероприятий

Веб-приложение для поиска, создания и участия в событиях. Позволяет пользователям находить интересные мероприятия в своем городе, создавать собственные события.

##  Функционал

### Реализовано:
- **Регистрация и аутентификация** пользователей
- **Создание событий** с детальной информацией (дата, место, категория, описание)
- **Участие в событиях** ("Пойду!" функционал)
- **Комментирование** событий
- **Избранное** - сохранение понравившихся событий
- **Рейтинги и отзывы** к событиям
- **Фильтрация** событий по категориям, дате, статусу
- **Поиск** по названию и описанию
- **Административная панель** для управления контентом
